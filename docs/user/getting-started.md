# 快速开始

欢迎使用 RokunTool! 本指南将帮助您快速上手使用这个基于 Electron 的插件化工具。

## 什么是 RokunTool?

RokunTool 是一个可扩展的桌面应用程序平台,允许通过插件来扩展功能。当前支持的插件包括:

- **微信分身插件** - 创建和管理多个微信分身
- **Rime 配置插件** - 管理 Rime 输入法配置 (开发中)

## 系统要求

### 必需环境

- **操作系统**: macOS 12.0 (Monterey) 或更高版本
- **Node.js**: 18.0.0 或更高版本
- **包管理器**: pnpm (推荐) 或 npm

### 检查环境

运行以下命令检查您的环境:

```bash
# 检查 Node.js 版本
node --version  # 应该 >= 18.0.0

# 检查 pnpm 是否安装
pnpm --version
```

如果 pnpm 未安装,请先安装它:

```bash
npm install -g pnpm
```

## 安装步骤

### 1. 克隆或下载项目

```bash
git clone <repository-url>
cd RokunTool
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 启动应用

```bash
pnpm dev
```

首次启动时,应用会:
- 初始化插件系统
- 扫描并加载可用插件
- 打开主窗口

## 基本使用

### 插件管理

1. **查看插件列表**
   - 在主界面左侧可以看到所有已安装的插件
   - 插件会显示状态:已启用 / 已禁用

2. **启用/禁用插件**
   - 点击插件卡片上的开关即可启用或禁用插件
   - 禁用的插件不会加载,不会占用系统资源

3. **配置插件**
   - 点击插件卡片可以进入插件详情页面
   - 在详情页面可以配置插件选项

### 权限管理

某些插件需要特殊权限才能正常工作:

- **文件写入权限** (`fs:write`) - 用于创建、修改文件
- **进程执行权限** (`process:exec`) - 用于执行外部程序

当插件需要权限时,会弹出请求对话框:
- 阅读权限说明
- 点击"允许"或"拒绝"
- 可以随时在设置中撤销已授予的权限

## 常用功能

### 微信分身

使用微信分身插件创建多个微信实例:

1. 确保已安装微信应用
2. 点击"创建分身"按钮
3. 授予必要的权限(文件写入、进程执行)
4. 等待创建完成
5. 在启动台找到新创建的微信分身(微信3、微信4等)

**注意事项**:
- 分身是独立应用,可以直接启动,不需要通过 RokunTool
- 分身数据由微信自己管理,插件不会干预
- 微信更新后,建议重建分身以避免兼容性问题
