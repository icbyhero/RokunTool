## Phase 1: 基础沙箱 (P0 - 必需)

**目标**: 防止插件直接访问 Node.js 模块

  - 定义禁止的全局对象

- [ ] 1.1.2 实现 createSandboxContext() 方法
  - 创建安全的沙箱上下文

  - 注入 exports, module, __dirname, __filename
  - 移除 require, process, global

- [ ] 1.1.3 实现 runInSandbox() 方法

  - 添加超时保护 (30秒)
  - 添加错误处理
  - 返回执行结果

- [ ] 1.2.1 修改 loadInstance() 方法

  - 删除 `import(mainPath)`

- [ ] 1.2.3 错误处理增强
